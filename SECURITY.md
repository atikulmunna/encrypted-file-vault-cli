# Security Policy

VaultCLI is under active development and is not yet ready for production use.

## Supported Versions

No stable release is currently supported.

## Reporting a Vulnerability

Until a dedicated security contact and release process are established, please avoid filing public issues that disclose exploitable details for unfinished security-sensitive code.

If a serious vulnerability is discovered after the project becomes usable, this policy will be updated with a private reporting channel and response process.
